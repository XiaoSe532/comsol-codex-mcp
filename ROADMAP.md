# Roadmap

## 0.1.x

- Improve Windows and Linux COMSOL discovery.
- Add more log diagnosis patterns.
- Add example stdout parsers.
- Add richer smoke test documentation for COMSOL-equipped machines.
- Add MCP client configuration examples for common hosts.

## 0.2.x

- Add optional project adapter loading from YAML.
- Add result parser plugin interface.
- Add Java API probe template generation.
- Add boundary/selection diagnostic templates.
- Add optional MPh backend experiments without making MPh mandatory.

## 0.3.x

- Add GUI bridge integration points for users who prefer COMSOL Desktop Java Shell workflows.
- Add long-running job monitor mode.
- Add restart/resume metadata for large parameter sweeps.
- Add structured provenance manifests for packaged outputs.

## Long-Term

- Cross-version compatibility matrix for COMSOL 5.x/6.x.
- Community recipe library.
- Domain-specific adapter packages that remain separate from the generic server.
