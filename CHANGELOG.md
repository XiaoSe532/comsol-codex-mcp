# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project uses semantic versioning after the first stable release.

## [0.1.0] - 2026-06-25

### Added

- Initial batch-first COMSOL MCP server.
- CLI command `comsol-tool`.
- COMSOL install discovery.
- Java API compile helper.
- COMSOL batch runner.
- Batch log inspector.
- COMSOL completion XML feature search.
- Lightweight output packager.
- Generated artifact cleaner.
- Built-in COMSOL agent workflow recipes.
- Minimal COMSOL Java API example.
- Documentation for architecture, recipes, safety, compatibility, quickstart, and tool reference.
